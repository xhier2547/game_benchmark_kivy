## Installation
 you should be install kivy before run , let do it !

  ### install kivy

  ``` bash
  pip install kivy
  ```

  ### Run App

   - **Windows:**

  ``` bash
  python main.py
  ```

  - **macOS/Linux:**

  ``` bash
  python3 main.py
  ```
